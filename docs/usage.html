Usage Guide

How to connect MCP clients and use the exposed tools.

Exposed Tools

The server exposes three tools via MCP. Any MCP-compatible client can discover and call them automatically.

get_schema
Returns a compact Markdown summary of the database schema — tables, columns, types, nullability, and primary keys.
execute_query
Runs a validated SELECT query inside a READ ONLY transaction with hard 50-row pagination.
simulate_query
Runs any SQL inside a transaction and rolls it back immediately — zero risk to persistent data.

Connecting MCP Clients

Claude Desktop

Add to your Claude Desktop config (claude_desktop_config.json):

{
  "mcpServers": {
    "db-guardrail": {
      "command": "npx",
      "args": [
        "db-guardrail-mcp",
        "--connectionString", "postgresql://...",
        "--apiKey", "your_key_here"
      ]
    }
  }
}

Custom CLI / Script

import { spawn } from 'node:child_process';

const server = spawn('npx', [
  'db-guardrail-mcp',
  '--connectionString', 'postgresql://...',
  '--apiKey', 'your_key_here',
]);

// Communicate over stdin/stdout using JSON-RPC 2.0

Important Behavior

Pagination is Hard-Capped at 50 Rows

Regardless of what the model or client requests, execute_query will never return more than 50 rows. This protects against runaway queries and context-window overflow. If your query would return more than 50 rows, the response includes a notice indicating truncation.

Data Never Leaves Your Machine

Your connection string, database schema, queries, and query results stay entirely on your machine. They are never sent to any hosted server. The only outbound call is a lightweight ping to validate your API key and log that the tool is active — no data, no queries, no results.

Mutations Require Simulation

execute_query runs inside a BEGIN READ ONLY transaction — any INSERT, UPDATE, DELETE, or DDL statement will be rejected at the database level. To preview the effects of a mutation, use simulate_query, which runs inside a normal transaction and issues a ROLLBACK after execution.
