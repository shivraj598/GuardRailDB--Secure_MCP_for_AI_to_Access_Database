Release notes and announcements.

v1.0.0 Initial Release
July 23, 2026
  • get_schema — Queries information_schema and compresses it into LLM-readable Markdown, cutting token usage for database context.
  • execute_query — Runs validated SELECT queries inside a READ ONLY transaction with hard 50-row server-side pagination.
  • simulate_query — Runs any SQL in a sandboxed BEGIN/ROLLBACK transaction so the model can preview mutations with zero risk.
  • Zod-based input validation on every tool call closes off SQL injection from malformed LLM output.
  • Version-change notification — prints a visible banner on startup when the package is updated.
  • Zero-install via npx db-guardrail-mcp.
